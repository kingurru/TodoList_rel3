Функционал приложения:

1. Задачи можно добавить в любой из блоков
2. Задачи можно отметить как выполненные
3. Выполненные задачи в любом из блоков можно скрыть нажатием на заголовок блока
4. Выполненные задачи можно удалить нажатием на иконку перекрестия напротив названия
5. Созданную задачу можно отредактировать двойным кликом левой кнопки мыши на наименовании
6. Применить редактирование можно также двойным кликом левой кнопки мыши или нажатием клавиши "Enter"
7. После перезагрузки или закрытия вкладки, задачи остаются в localStorage. Таким образом можно использовать актуальные данные приложения в любое время
